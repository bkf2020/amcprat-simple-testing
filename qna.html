Question and answers for the simple AMC Practice Tool.

What is this version of AMCPRAT?

This is a simple AMC Practice Tool. It is designed to have only HTML and CSS (for the web site). Here is the link to the full-featured AMCPRAT. Python is used to scrape the AMC Problems and to generate the problemsets.

How do I use it?

Select one of the links above for your problemset. You can either select AIME, AMC 8, AMC 10, or AMC 12 problems. The difficulty you choose can be Easy, Medium, Hard, or Very Hard.

Are problems updated daily?

Yes. These problems are updated each day. It is scheduled to update at 00:00 UTC, but usually happens 30 or so minutes after that. If you want to save your problem set, please bookmark the links to the problems.

What are the difficulties for the problemsets?

Difficulty for AIME:

  • Easy: Problems 1 - 5
  • Medium: Problems 6 - 10
  • Hard: Problems 11 - 13
  • Very Hard: Problems 14 - 15

Difficulty for AMC8, AMC10, AMC12:

  • Easy: Problems 1 - 15
  • Medium: Problems 16 - 18
  • Hard: Problems 19 - 22
  • Very Hard: Problems 23 - 25

How do I report an issue with the code, a math problem, etc.?

Go onto the Github repo and open a new issue. Be sure to place a link to where you found the problem if it is on the website or the location of the code. I plan to open an email address soon...

Can I check the problems for accuracy?

Go to the testing page to check the problems.

The math equations/diagrams are hard to see.

Please zoom in on your web browser, so it becomes clearer.

Where can I check known issues?

View the issues page on Github.

What does the code do?

First head to the Github repo. In the "scraper" directory, "scraper.py" is used to create the websites for every problem. In the "problemsets" directory, the problemsets are there, and so are all the problems in the system. Lastly, the ".github/workflows" folder contains the workflow that runs a update.py program each day to create the problemsets.
